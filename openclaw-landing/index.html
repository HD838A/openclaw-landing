什么是 OpenClaw？

OpenClaw 是一个自托管网关，它连接你喜欢的聊天应用 —— WhatsApp、Telegram、Discord、iMessage 等 —— 与 AI 编程助手。你只需在自己的电脑（或服务器）上运行一个 Gateway 进程，它就成为你的消息应用与随时可用的 AI 助手之间的桥梁。

它适合谁？想要拥有私人 AI 助手、但又不想放弃数据控制权或依赖托管服务的开发者与高级用户。

OAuth 认证

安全的模型认证

OpenClaw 支持通过 OAuth 进行「订阅认证」，适用于提供 OAuth 服务的提供商（特别是 OpenAI Codex (ChatGPT OAuth)）。

🔐 OAuth 令牌交换

支持 PKCE 安全流程，自动处理令牌刷新，无需手动管理

📁 多账户支持

通过 auth-profiles.json 支持多个账户，可按会话路由到不同配置文件

🔑 密钥安全存储

按代理存储认证信息，支持 API 密钥和 OAuth 令牌

⚡ 自动刷新

令牌自动刷新，运行时自动处理过期时间

支持的认证方式

# OpenAI Codex OAuth 登录
openclaw models auth login --provider openai-codex

# Anthropic 设置令牌（订阅认证）
openclaw models auth setup-token --provider anthropic

# 查看认证状态
openclaw models status

⚠️ 注意

Anthropic 的设置令牌支持仅指技术兼容性，请自行查看 Anthropic 当前的条款。对于生产环境，API 密钥认证比订阅设置令牌认证更安全。

快速开始

5 分钟内启动

1

安装 OpenClaw

使用 npm 全局安装

npm install -g openclaw@latest
2

初始化并安装服务

运行引导程序并安装守护进程

openclaw onboard --install-daemon
3

登录渠道并启动 Gateway

选择你想使用的通讯渠道

# 登录渠道
openclaw channels login
# 启动 Gateway
openclaw gateway --port 18789
4

打开控制台

在浏览器中打开控制 UI 开始使用

http://127.0.0.1:18789/
