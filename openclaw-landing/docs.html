📚 文档

快速上手 OpenClaw，打造你的私人AI助手

🚀 快速开始

在 5 分钟内启动 OpenClaw 开始使用。

前置要求

📦

Node.js 22+

🔑

AI 模型 API Key

⏱️

5 分钟时间

步骤 1：安装 OpenClaw

npm install -g openclaw@latest

步骤 2：初始化并安装服务

运行引导程序，按照提示完成配置，并安装守护进程

openclaw onboard --install-daemon

步骤 3：登录通讯渠道

选择你想使用的通讯渠道（WhatsApp、Telegram、Discord 等）

openclaw channels login

步骤 4：启动 Gateway

openclaw gateway --port 18789

步骤 5：打开控制台

在浏览器中打开控制 UI 开始使用

http://127.0.0.1:18789/

💡 提示：可以使用 openclaw start 命令一键启动所有服务

⚙️ 配置说明

配置文件位置

配置文件位于 ~/.openclaw/openclaw.json

基本配置示例

{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123"]
    }
  },
  "messages": {
    "groupChat": {
      "mentionPatterns": ["@openclaw"]
    }
  }
}

安全设置

默认配置下任何人都可以访问，建议配置白名单：

  • channels.whatsapp.allowFrom - 允许的白名单号码
  • groups.requireMention - 群组中是否需要 @提及
